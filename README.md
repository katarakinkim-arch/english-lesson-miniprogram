# 高中英语教案小程序（人教版 2019 · 必修一二册）

把原来的单文件 HTML 教案库，转成了**可以直接在微信里运行的真实小程序**。

## 一、怎么打开运行（无脑三步）

1. 电脑装好 **微信开发者工具**（官网 weixin.qq.com 下载）。
2. 打开开发者工具 → 「导入项目」→ 目录选 **本文件夹**（`miniprogram/`）。
3. AppID 选 **测试号**（或「游客模式」也行，已配置为 `touristappid`）→ 点导入即可预览。

> 不需要自己装任何 npm 包、不需要服务器、不需要后端。教案数据已经内置在工程里。

## 二、功能

| 页面 | 说明 |
|------|------|
| 教案库（首页） | 搜索 + 按「必修一/必修二」筛选 + 按「阅读/语法/写作」课型筛选，卡片列表 |
| 教案详情 | 完整教案内容（目标/重难点/过程/板书/练习/反思），底部可一键下载 **Word / PDF / PPT** |
| 我的下载 | 本地记录你下载过的教案，点开可再看，可一键清空 |
| 最近浏览 | 自动记录你打开过的教案，点开可再看，可一键清空 |

- 共 **30 课** 教案（必修一/二 各 15 课，每册阅读/语法/写作 各 10 课）。
- 下载的 Word/PPT/PDF 文件由小程序**在手机端现场生成**，再用系统预览打开，无需联网取文件。

## 三、文件结构

```
miniprogram/
├── app.js / app.json / app.wxss      # 全局配置、数据、样式（深蓝 TDesign 风格）
├── project.config.json               # 开发者工具配置（已设 touristappid）
├── sitemap.json
├── data/lessons.js                   # 30 课教案数据（内置）
├── utils/docgen.js                   # 端上生成 Word/PPT/PDF 的核心模块
├── pages/
│   ├── index/      # 教案库
│   ├── detail/     # 教案详情 + 下载
│   ├── downloads/  # 我的下载
│   └── recent/     # 最近浏览
└── images/tabbar/  # 底部导航图标
```

## 四、已知需要真机/模拟器验证的点

- **Word / PPT 生成**：纯前端 OOXML + ZIP 编码，逻辑已通过 Node 语法与结构校验，DevTools 里即可验证。
- **PDF 生成**：用 canvas 绘制中文后嵌入真 PDF。依赖 `wx.createSelectorQuery()` 取 canvas 节点，
  建议在**微信模拟器或真机**上点一次「下载 PDF」确认效果（详情页已预留隐藏 canvas）。

## 五、包大小

当前工程约 **502 KB**，远低于微信主包 2 MB 上限，后续若加更多册数也可轻松放进分包。
