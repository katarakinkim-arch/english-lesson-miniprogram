# 高中英语教案小程序（人教版 2019 · 必修一二册）

把原来的单文件 HTML 教案库，转成了**可以直接在微信里运行的真实小程序**，并接入了**微信云开发**做用户数据（下载 / 最近浏览 / 收藏 / 昵称头像）的云端记录，支持多设备同步。

## 一、本地快速预览（无脑三步，无需云）

1. 电脑装好 **微信开发者工具**（官网 weixin.qq.com 下载）。
2. 打开开发者工具 → 「导入项目」→ 目录选 **本文件夹**（`miniprogram/`）。
3. AppID 选 **测试号**（或「游客模式」也行，已配置为 `touristappid`）→ 点导入即可预览。

> 未开通云开发时，用户数据会**自动降级到本机存储**，功能照常可用，只是不能跨设备同步。

## 二、接入云开发（上线 / 多设备同步必须）

1. 在 [微信公众平台](https://mp.weixin.qq.com) 注册一个**真实小程序 AppID**（个人号即可），导入项目时填这个 AppID（不要用游客模式）。
2. 开发者工具顶部点 **「云开发」** → 开通 → 复制**环境 ID**。
3. 打开 `config.js`，把 `ENV_ID` 的 `'your-cloud-env-id'` 换成你的环境 ID。
4. 在左侧文件树 **右键 `cloudfunctions/user` → 上传并部署：云端安装依赖**。
   （只需部署这一个云函数，它已包含登录、资料、行为记录全部逻辑。）
5. 重新编译运行。首次打开会自动创建用户记录；之后下载/收藏等都会写入云端。

### 数据库集合（自动建，无需手动）
云开发控制台 → 数据库 → 新建集合 `users`，权限设为「仅创建者可读写」（默认即可）。
文档结构：`{ _openid, nickName, avatarUrl, downloads:[{lessonId,format,time}], recents:[lessonId], favorites:[lessonId], createdAt }`

## 三、功能

| 页面 | 说明 |
|------|------|
| 教案库（首页） | 搜索 + 按「必修一/必修二」筛选 + 按「阅读/语法/写作」课型筛选，卡片列表 |
| 教案详情 | 完整教案内容（目标/重难点/过程/板书/练习/反思），底部可一键下载 **Word / PDF / PPT**，并支持 ☆ 收藏 |
| 我的下载 | 云端记录下载过的教案（含格式与 time），点开可再看，可一键清空 |
| 最近浏览 | 云端自动记录打开过的教案，点开可再看，可一键清空 |
| 我的（个人中心） | 头像/昵称（合规授权获取）、下载/最近/收藏统计、收藏列表管理、清空全部数据 |

- 共 **30 课** 教案（必修一/二 各 15 课，每册阅读/语法/写作 各 10 课）。
- 下载的 Word/PPT/PDF 文件由小程序**在手机端现场生成**，再用系统预览打开，无需联网取文件。
- 用户数据通过 `utils/clouduser.js` 统一读写：**云端优先，失败自动降级本地**。

### 基础框架标配能力（已接入）
- **分享**：首页可转发整个小程序给好友；详情页可转发具体教案（含标题/路径），并支持**分享到朋友圈**（`onShareTimeline`）；详情页底部有可见「📤 分享」按钮（`open-type="share"`）。
- **下拉刷新**：教案库 / 我的下载 / 最近浏览 三个列表页均已开启 `enablePullDownRefresh`，下拉即重新加载。
- **错误态 + 重试**：下载 / 最近页网络失败（或云未就绪）时显示「数据加载失败 + 点击重试」，不再静默空白。
- **随页面切换刷新**：下载 / 最近页在 `onShow` 时自动刷新，从详情页返回即同步最新收藏/下载状态。


## 五、隐私合规（提交审核前必看）

- 昵称/头像使用微信**合规方式**获取（`button open-type="chooseAvatar"` + `input type="nickname"`），
  不调用已废弃的 `getUserProfile`，符合 2023 年后规范。
- 上线前在 **微信公众平台 → 设置 → 服务内容声明 / 用户隐私保护指引** 中，
  填写《隐私保护指引》并声明收集「昵称、头像」与「收藏/下载记录」的用途。
- 若后台提示需开启隐私弹窗，可在 `app.json` 增加 `"__usePrivacyCheck__": true`（默认已按最小必要原则实现）。

## 六、已知需要真机/模拟器验证的点

- **Word / PPT 生成**：纯前端 OOXML + ZIP 编码，逻辑已通过 Node 语法与结构校验，DevTools 里即可验证。
- **PDF 生成**：用 canvas 绘制中文后嵌入真 PDF。依赖 `wx.createSelectorQuery()` 取 canvas 节点，
  建议在**微信模拟器或真机**上点一次「下载 PDF」确认效果（详情页已预留隐藏 canvas）。
- **云函数联调**：部署 `cloudfunctions/user` 后，用真实 AppID 在模拟器/真机点一次下载或收藏，
  到云开发控制台 `users` 集合确认记录写入成功。

## 七、包大小

当前工程约 **502 KB**，远低于微信主包 2 MB 上限，后续若加更多册数也可轻松放进分包。
